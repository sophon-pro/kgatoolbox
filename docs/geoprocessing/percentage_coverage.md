## Goal

Calculate the `percentage coverage` of sub-feature classes within the boundary of the `main feature class`.

## Summary

This tool is used to calculate the coverage percentage of the input feature class(es) over any boundary.

![point to polygon ui](../assets/percent_coverage_ui.png)
## Illustration

![point to polygon](../assets/)

## Usage Note

* This tool supports multiple input feature class which coverage percentage over any boundary will be calculated.
* Coverage Percentage will be calculated based on the total area of each input feature classes over the total area of the boundary.

## Parameters

This tool has some importances parameters as show in the table below.

| Parameter | Explanation | Data Type |
|:---------|:------------|:----------|
| Input_Feature | Feature classes for which coverage percentage over a boundary will be calculated. | GPMultiValue |
| Input_Boundary_Feature | Specifies the boundary feature class. | GPFeatureLayer |
